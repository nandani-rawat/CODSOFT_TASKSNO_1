# 🧹 Task 1 - Data Cleaning & Preprocessing

This project was completed as part of the **CodSoft Data Analytics Internship**. The objective of this task is to clean and preprocess the Superstore Sales dataset using Python and Pandas.

---

## 📌 Objective

The objective of this project is to clean and preprocess the dataset by:

- Importing and inspecting the dataset
- Identifying missing values
- Removing duplicate records
- Correcting inconsistent data entries
- Handling null values
- Preparing the dataset for further analysis
- Saving the cleaned dataset as a new CSV file

---

## 📂 Dataset

**Dataset Name:** Superstore Sales Dataset

**Format:** CSV

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 📋 Tasks Performed

- Imported the dataset
- Explored the dataset structure
- Checked for missing values
- Removed duplicate records
- Handled missing values
- Corrected data types where required
- Cleaned inconsistent data entries
- Saved the cleaned dataset as a new CSV file

---

## 📁 Project Files

- `Data_Cleaning.ipynb` – Jupyter Notebook containing the complete implementation
- `Superstore.csv` – Original dataset
- `Cleaned_Superstore.csv` – Cleaned dataset
- `README.md` – Project documentation

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open `Data_Cleaning.ipynb` in Jupyter Notebook.
3. Run all the cells in sequence.
4. The cleaned dataset will be generated and saved as a new CSV file.

---

## 🎯 Outcome

The Superstore Sales dataset was successfully cleaned by handling missing values, removing duplicate records, correcting inconsistencies, and preparing the data for further analysis.
